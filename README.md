# LOMLOE Agentic RAG - WebGPU Local AI

Esta aplicación funciona como un entorno IA **100% privado y autónomo** operando exclusivamente dentro de tu navegador web usando `Transformers.js` y `WebGPU`. No depende de servidores locales como Ollama ni envía tus documentos a APIs externas en la nube.

## Cómo funciona
1. **Modelos Locales**: Utiliza modelos de la familia **Gemma 4** o **Qwen** ligeros.
2. **Motor de Vectores**: Convierte documentos PDF en bases de datos de conocimiento puramente utilizando tu tarjeta gráfica.
3. **Pipeline Standalone**: Solo requiere subir los archivos de este repositorio directamente a un *Hugging Face Space* configurado como `Static HTML`.
